## 🎯 Add Your Photo (3 Easy Steps)

### Step 1: Prepare Your Image
- **Format:** JPG, PNG, or WebP
- **Size:** 400x400 pixels (or any square ratio)
- **File size:** Keep it under 500KB for fast loading
- **Quality:** Use a clear, professional headshot

### Step 2: Add the File
Save your photo as **`profile.jpg`** in this folder:
```
D:\portfolio\frontend\public\images\profile.jpg
```

**Alternative names supported:**
- `profile.png`
- `profile.jpeg`
- `avatar.jpg`

### Step 3: Restart the Dev Server
```bash
cd D:\portfolio\frontend
npm run dev
```

That's it! Your photo will now appear in the About section.

## 🖼️ Image Optimization Tools

Want to optimize your image before adding?

**Free Online Tools:**
- [TinyPNG](https://tinypng.com/) - Compress PNG/JPG
- [Squoosh](https://squoosh.app/) - Google's image optimizer
- [ILoveIMG](https://www.iloveimg.com/) - Resize and compress

**Recommended Settings:**
- Format: JPG (85% quality)
- Dimensions: 400x400px
- File size: < 200KB

## 🔍 Troubleshooting

**Photo not showing?**
1. Make sure the file is named exactly `profile.jpg`
2. Check the file path: `frontend/public/images/profile.jpg`
3. Restart the dev server
4. Clear browser cache (Ctrl+Shift+R)

**Photo looks stretched?**
- Use a square image (1:1 aspect ratio)
- Recommended: 400x400 pixels

**File size too large?**
- Use TinyPNG or Squoosh to compress
- Aim for < 200KB
