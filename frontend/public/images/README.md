# Profile Picture Instructions

## How to Add Your Profile Picture

1. **Get a professional photo of yourself** (recommended size: 400x400px or 500x500px)

2. **Save the image** in this folder (`frontend/public/images/`) with one of these names:
   - `profile.jpg` (recommended)
   - `profile.png`
   - `avatar.jpg`
   - `shoaib.jpg`

3. **Supported formats:** JPG, PNG, WebP, GIF

4. **Recommended specs:**
   - Square aspect ratio (1:1)
   - Minimum 200x200px
   - File size under 500KB for fast loading

## Example:
If you save your photo as `profile.jpg` in this folder, it will be accessible at:
`/images/profile.jpg`

The About section will automatically use it once you add the image file.

---

**Quick Tip:** Use a clear, professional headshot with good lighting for the best impression!
